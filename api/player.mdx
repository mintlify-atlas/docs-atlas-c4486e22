---
title: "Player"
description: "Main entry point to the TextAlive App API. Loads song maps, lyrics, and timing data, and provides APIs for multimedia performance synchronized with music playback."
---

`Player` is the primary class in the TextAlive App API. It implements the [`IPlayer`](#iplayer-interface) interface and serves as the single entry point for loading songs, managing playback, and querying musical data.

```typescript
import { Player } from "textalive-app-api";

const player = new Player({
  app: { token: "your-developer-token" },
});
```

### isVideoSeeking


- [`onTimeUpdate`](/api/player-events#ontimeupdate) events are suppressed.
- [`mediaPosition`](#mediaposition) continues to update without triggering new render frames.

## Usage example

```typescript
import { Player, PlayerListener, IVideo, Timer } from "textalive-app-api";

const player = new Player({
  app: { token: "your-developer-token" },
  mediaElement: document.getElementById("media"),
});

const listener: PlayerListener = {
  onAppReady(app) {
    if (!app.managed) {
      // No host connected — load a song directly
      player.createFromSongUrl("https://piapro.jp/t/hZ35/20240130103028");
    }
  },

  onVideoReady(video: IVideo) {
    console.log("Video ready. Duration:", video.duration, "ms");
  },

  onTimerReady(timer: Timer) {
    console.log("Timer ready:", timer);
  },

  onTimeUpdate(position: number) {
    // Called every render frame
    const beat = player.findBeat(position);
    if (beat) {
      const progress = beat.progress(position); // 0.0 → 1.0 within the beat
      console.log(`Beat ${beat.index}, progress: ${progress.toFixed(2)}`);
    }
  },

  onPlay() {
    console.log("Playback started");
  },

  onStop() {
    console.log("Playback stopped");
  },
};

player.addListener(listener);
```
