---
title: "Ease"
description: "A collection of easing functions for smooth animation, inherited from the CreateJS open source library."
---

The `Ease` class provides a collection of static easing functions you can use to create smooth, non-linear animations synchronized with music playback. All functions accept an input in the range `[0, 1]` and return an output in the same range.

This implementation is inherited from the [CreateJS](http://createjs.com/) open source library. See the [live demo](https://www.createjs.com/demos/tweenjs/tween_sparktable) to visualize all available easing curves.

<Tip>
Combine any easing function with a rendering unit's `progress(position)` method to drive smooth animations. `progress` maps the current playback position to a `[0, 1]` value within the unit's time range, and you pass that value into an easing function to get a non-linear result.
</Tip>

## Practical example

```typescript
import { Ease } from "textalive-app-api";

// In onTimeUpdate:
const beat = player.findBeat(position);
if (beat) {
  const t = Ease.cubicOut(beat.progress(position));
  element.style.transform = `scale(${1 + t * 0.3})`;
}

// Custom power ease:
const strongEase = Ease.getPowInOut(4);
const t = strongEase(char.progress(position));
```

## Linear

<ResponseField name="Ease.linear" type="(t: number) => number">
  A linear (no-op) easing function. Output equals input.
</ResponseField>

## Flash-style ease

<ResponseField name="Ease.get" type="(amount: number) => (input: number) => number">
  Returns an easing function that mimics the simple -100 to 100 easing in Adobe Flash/Animate.

  <Expandable title="Parameters">
    <ParamField path="amount" type="number">
      A value from `-1` (ease in) to `1` (ease out) indicating the strength and direction of the ease.
    </ParamField>
  </Expandable>
</ResponseField>

## Power eases

## Back eases

<ResponseField name="Ease.getBackIn" type="(amount: number) => (input: number) => number">
